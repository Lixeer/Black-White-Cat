>基于分布式服务器概念而设立的一款聊天应用

建议使用 **Python3.10** 最佳


## 1 简介

- 完全开源的一款聊天应用
- 真正意义上安全自由的聊天工具
- 所有服务器将由自己架设,没有意义上的官方服务器

## 2 概念
>在这款应用里，服务器将不知道你是谁,你可以决定你叫什么名字,也可以重名。你加入的每一个聊天都是不同的服务器，只要有密钥（群聊邀请连接）你就可以加入群聊。只要有需要，就可以自己搭建服务器以保证绝对安全
